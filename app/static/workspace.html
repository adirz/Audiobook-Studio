    QA Scoring

    The QA transcribes generated audio with Whisper (STT) and compares it word-by-word to the original text. Common STT equivalences are normalized before comparison: contractions are expanded ("I'm" → "I am", "we'll" → "we will"), digits are converted to words ("150" → "one hundred fifty"), and expected pronunciation substitutions are ignored.

    The Similarity threshold is the minimum adjusted match ratio for a chunk to pass (default 0.95 = 95%). Type a value or use the arrows — 0.005 increments let you tune precisely.

    ML mode will use a speaker-embedding model to verify the generated voice matches the one selected, in addition to the text match check.
